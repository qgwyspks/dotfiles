# Hyprland

## Hyprland Dependencies

- Window Manager Hyprland
- Shell Zsh + zinit + starship
- Terminal Kitty
- Panel Waybar
- Notify Daemon Dunst
- Launcher Rofi
- File Manager Dolphin Ranger Thunar
